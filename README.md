# VoiceAI Studio

Local-first Windows desktop application for voice AI workflows built with Electron, React, Tailwind, and Python.

## Workspaces

- `apps/desktop` - Electron main, preload, and React renderer
- `backend` - Python AI runtime for training, inference, separation, and audio processing
- `packages/shared` - shared IPC contracts, schemas, and constants
- `packages/ui` - reusable React UI primitives

## Core Areas

- model downloads
- dataset management
- training
- inference
- audio processing
- download queue
- settings
